Installation
============

.. note::
  This project is still in it's beta test phase. The project is currently not up on PyPI, so the below
  installation methods will not work. To install a test version, please run the following ``pip`` command
  in a virtual environment::
    pip install --index-url https://test.pypi.org/simple/ custconsole

This project is published through PyPI, so the preferred installation method is with ``pip``.

Windows
^^^^^^^
.. code-block::

  pip install --user custconsole

MacOS / Linux
^^^^^^^^^^^^^
.. code-block::

  pip install custconsole
